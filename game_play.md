How the Game Works
🖍️ 1. Start the Game

Visit the website and enter a nickname.

Select a language for the word list.

Choose to:

Play! — join a random public game, or

Create Private Room — make a room to play with friends and customize settings.

🕹️ Gameplay Rules
🎯 2. Turns and Rounds

The game is divided into multiple rounds.

In each round, one player becomes the drawer while others are guessers.

✏️ 3. Drawing

The drawer gets a choice of words (usually three) and picks one to draw.

They must illustrate the word using the game’s simple paint tools (brush, eraser, color palette, etc.).

Important: You cannot draw letters or numbers to spell the word — only pictures count.

💡 4. Guessing

All other players type their guesses in the chat box as the drawing unfolds.

The game shows blanks and periodically reveals letters as hints while time counts down.

Fast, correct guesses earn more points.

🏆 5. Scoring

Guessers earn points based on how quickly they guess the correct word.

The drawer also earns points for each correct guess by others.

After all rounds finish, the player with the highest total score wins.

🛠️ Customization Options (Private Rooms)

If you create your own game room, you can adjust:

Number of rounds

Draw time per round (e.g., 30–180 seconds)

Word hints and letter reveal options

Word count and word mode

Custom word lists (especially fun with friends)

Player limits and language choice

You can send friends a link to join your private room instantly.
