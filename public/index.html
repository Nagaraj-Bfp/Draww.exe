How to play

1. Set up — Pick your character, enter your name, then create a room or join one with the link.

2. Lobby — Wait for everyone to join. The host starts the game when ready.

3. Rounds — Each round one player is the drawer; others are guessers.

4. Drawing — The drawer gets a secret word and sketches it on the whiteboard. No letters or numbers — only pictures.

5. Guessing — Guessers type in the chat. Blanks and hints appear as time runs. Faster correct guesses score more points.

6. Scoring — Guessers earn points for speed; the drawer earns points for each correct guess. Highest total after all rounds wins.
